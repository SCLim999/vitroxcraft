## 1. 系统需求

| 项目 | 要求 |
|---|---|
| 设备 | 电脑（键盘鼠标，体验最佳）或**手机/平板（触屏操作）** |
| 浏览器 | Chrome、Edge、Firefox、Safari 的近年版本 |
| 网络 | 打开页面时需要联网（加载 Three.js 图形库和多人组件）；单人游玩加载完成后即可离线 |
| 安装 | **无需安装任何东西** —— 就是一个网页 |

## 2. 如何开启

### 方式 A：在线访问（推荐，适合发给学生）

仓库合并到 `main` 并开启 GitHub Pages 后，直接访问：

```
https://sclim999.github.io/pps2114-quiz/minecraft/
```

> 开启 Pages 的方法：仓库 **Settings → Pages** → Source 选 **Deploy from a branch** → 分支选 `main`、目录选 `/ (root)` → Save，等一两分钟即可。

### 方式 B：本地打开

1. 下载 `minecraft/index.html` 这一个文件（GitHub 页面上点 **Download raw file**）；
2. 双击用浏览器打开即可。

打开页面后会先看到一个**密码锁屏**，输入正确密码并点「解锁」（或按 Enter）才能看到开始菜单——每次重新打开页面都需要再输入一次（密码请向管理者索取；源码里只存了密码的 SHA-256 哈希值，并不是明文）。解锁后，点绿色的 **「▶ 进入园区」** 按钮开始。浏览器会锁定鼠标指针（这是正常的，按 `Esc` 可随时退出）。

## 3. 操作方法

| 按键 / 操作 | 功能 |
|---|---|
| `W` `A` `S` `D` | 前 / 左 / 后 / 右移动 |
| 移动鼠标 | 转动视角 |
| `空格` | 跳跃（上台阶、上坡道时会用到） |
| `F` | 切换**飞行模式**（空格上升、`Shift` 下降）——想俯瞰全景或上屋顶时非常好用 |
| 鼠标**左键** | 破坏方块 |
| 鼠标**右键** | 放置当前选中的方块 |
| `1`–`9` 或 滚轮 | 切换物品栏方块 |
| `E` | 靠近金色信息方块时：阅读建筑知识；弹窗打开时：关闭弹窗 |
| `Q` | 显示 / 隐藏任务清单 |
| 弹窗里的 🔊 按钮 | 朗读当前这段讲解/对话（再点一下或按 ⏹ 可停止） |
| `M` | 开关音效 |
| `Esc` | 解锁鼠标，回到菜单（再点「进入园区」继续） |

左上角显示你的坐标，下方是**小地图**——固定比例的园区俯视图，圆圈是中央庭院方位参照，金色/绿色小点是未探索/已探索的知识点，浅蓝小点是 NPC，白色三角箭头是你自己（会随转身方向旋转），方便随时确认自己在园区哪个角落、该往哪边走。顶部显示探索进度、**⏱ 游玩计时**（只在实际操控角色时累计，开菜单或看弹窗时会暂停）和在线人数。

**📱 手机 / 平板（触屏操作）**：点「进入园区」后自动切换触屏模式——**左侧摇杆**移动、**右侧空白处滑动**转视角，右下角按钮：`✈` 飞行开关、`⬆` 跳跃/上升、`⬇` 下降（飞行时）、`⛏` 挖方块（按住连挖）、`🧱` 放方块、`💬` 对话/阅读信息点；右上角 `☰` 返回菜单。建议**横屏**游玩。

## 4. 玩法一：建筑探索任务

园区里藏着 **10 个金色信息方块**（上方有蓝色 ⓘ 标记漂浮）。走到方块附近（约 3 格内），屏幕下方会出现提示，按 `E` 即可阅读该处的建筑设计讲解（中英对照标题）。按 `Q` 可随时查看还有哪些没找到。

| # | 知识点 | 在哪里找 |
|---|---|---|
| ① | 中央圆形庭院 | 园区正中心，水池旁 |
| ② | 无门的入口 | 南面大门外的广场（出生点沿步道直走） |
| ③ | 环形景观坡道 | 入口西侧的草坪上（讲解设计概念） |
| ④ | 绿色屋顶花园 | 环形楼屋顶——走**庭院西侧深灰色楼梯塔**（内部楼梯直通屋顶花园） |
| ⑤ | 几何体量的碰撞 | 东侧白色大楼（VITROX 招牌楼）西南角外 |
| ⑥ | 工业 × 人文校园 | VITROX 大楼内部一层（从面向庭院的开口进入） |
| ⑦ | Campus 3.0 与鱼池庭院 | 沿东侧马路走到新园区，鱼池木桥南端 |
| ⑧ | 启发 ViTrox 的相机 | ViTrox Gallery 走廊中段的金色展台（玻璃罩里的相机） |

| ⑩ | 电脑室 · CodingMine 编程课堂 | VITROX 招牌大楼四楼电脑室，两室分隔墙的大黑板旁 |

**🔊 文字转语音（Text-to-Speech）**：每个知识点和 NPC 对话弹窗右下角都有一个朗读按钮，点一下就用浏览器内建的语音朗读当前显示的文字（跟随所选语言：中文/English/Bahasa Melayu）。也可以在开始菜单勾选「自动朗读讲解内容」，之后每次打开新的知识点或对话都会自动开始朗读——适合朗读能力较弱的学生，或当作背景讲解使用。此功能不需要联网、不需要账号，由浏览器自带的语音引擎驱动，语音音色因浏览器/系统而异；如果浏览器不支持，朗读按钮和开关会自动隐藏。

**🚻 男声 / 女声**：NPC 对话会自动按角色造型选择声音——男性角色（保安、工程师、创办人等）用男声，女性角色（Siti、Mei Ling、Priya 等）用女声。建筑知识点讲解没有固定角色，可以在开始菜单的「讲解旁白声音」处选择 👨 或 👩，之后所有知识点朗读都会用你选的声音。声音的实际音色取决于你电脑/浏览器安装了哪些语音包（Windows/Edge 通常自带中英文男女声，Chrome 在部分系统上语音选择较少）。

正门进来的门厅还有一座白色接待柜台（Siti 就站在柜台后）和一座横跨门厅上空的二楼玻璃连桥（垂吊绿植 + Welcome 横幅），连接环形楼两侧的二层。

VITROX 招牌大楼**四楼**东半层有**两间电脑室**，共约 20 台电脑（书桌 + 显示器 + 主机）；从食堂旁的蓝色地毯走进楼梯间，沿直跑楼梯一路跳上四楼即可到达（记得边走边按空格跳上台阶）。

集齐 10 个会有通关祝贺。每个知识点都对应真实建筑的设计概念：放射状布局、实与虚、旋转动感、绿色建筑、几何并置、工业与校园文化的融合，以及电脑室里的编程教育。

园区里还住着 **20 位 NPC 人物**（保安、接待员、工程师、园丁、厨师、学生、职员、建筑师、讲师、市场专员），分布在大门、庭院、屋顶、食堂、球场、停车场、Campus 3.0 鱼池边、ViTrox Education 和四楼电脑室。走近时他们会转身看你，按 `E` 可以交谈——对白也支持三种语言，还会给你指路。三位 ViTrox 创办人（Chu Jenn Weng、Steven Siaw、Yeoh Shih Hoong）平时会在庭院附近的建筑物之间走动、四处溜达，不会一直固定站在原地——多逛逛，说不定能在不同的地方遇见他们。

## 5. 玩法二：自由建造

物品栏提供 9 种方块：**草方块、石头、白色混凝土、玻璃幕墙、木板、原木、树叶、ViTrox 蓝、太阳能板**——正是搭建这座建筑所用的「材料」。你可以：

- 给园区加盖新楼、天桥、雕塑；
- 用「ViTrox 蓝」在墙上拼字；
- 拆掉一面墙看看楼板和柱网结构（金色信息方块不可破坏）。

你的所有改动会自动保存在本机浏览器里（详见第 8 节）。

## 7. 多人连线

无需任何服务器，基于浏览器点对点连接（WebRTC），适合课堂上十几人同时游玩。

### 房主（老师/发起人）

1. 在开始菜单填好**你的名字**；
2. 点 **「创建多人房间」**，几秒后会显示一个 **5 位房间代码**（如 `AB3CD`）；
3. 把代码告诉同学；
4. 点「进入园区」开始玩。**房主的页面必须保持打开**——房主就是服务器。

### 加入者（学生）

1. 打开同一个网页，填好**你的名字**；
2. 在「房间代码」框输入房主给的 5 位代码，点 **「加入房间」**；
3. 看到「✅ 已加入房间」后，点「进入园区」。

### 连线后

- 每个人都是一个方块小人，头顶有名字牌，位置实时同步；
- 任何人挖 / 放方块，所有人都会看到；
- 顶部显示当前在线人数和房间代码；
- 有人加入或离开会有提示。

### 多人模式须知

| 事项 | 说明 |
|---|---|
| 世界归属 | 世界以**房主**的存档为准；加入者的改动只存在房主那里，不会写进自己浏览器 |
| 重置世界 | 只有房主可以重置，重置会同步给所有人 |
| 房主关页面 | 所有人断线（世界仍保留在房主浏览器的存档中，重开房间即可继续） |
| 探索进度 | 每人独立计算，互不影响 |
| 人数建议 | 建议 ≤ 15 人一个房间；人多可开多个房间 |
| 连不上？ | 多人连线使用公共 PeerJS 云做「牵线」，个别校园网防火墙可能拦截。可换网络（如手机热点）重试 |

> **进阶**：学校若想自建信令服务器（如内网使用），可在网址后加参数指定：
> `index.html?peerhost=服务器IP&peerport=9000&peerpath=/`（服务器端运行开源的 [peerjs-server](https://github.com/peers/peerjs-server) 即可）。

## 8. 存档与重置世界

- **自动存档**：你放置/破坏的每一个方块都会自动记录到本机浏览器的 `localStorage`，下次打开同一浏览器会自动恢复。
- **换浏览器/换电脑不带存档**；清除浏览器数据会清掉存档。
- **重置世界**：按 `Esc` 回菜单 → 点「🔄 重置世界」→ 确认后恢复原始园区（多人模式下仅房主可用）。

## 9. 常见问题（FAQ）

**打开后一直黑屏 / 停在菜单没反应？**
需要联网加载 Three.js 图形库。检查网络后刷新（`Ctrl+F5` 强制刷新）。

**点「进入园区」没锁定鼠标？**
个别浏览器需要再点一次页面；确保没有其他弹窗。`Esc` 后想继续，回菜单再点「进入园区」。

**画面卡顿？**
关闭其他标签页；浏览器硬件加速要开启（设置 → 系统 → 使用硬件加速）。

**掉进洞里 / 卡在方块里？**
按 `F` 开飞行飞出来；掉出世界会自动传送回出生点。

**「创建多人房间」显示组件未能加载？**
多人组件（PeerJS）需联网加载，且部分内网会拦截。刷新重试或换网络。

**找不到房间？**
确认代码输入正确（5 位、不分大小写）、房主页面还开着、双方都能上网。

**手机能玩吗？**
可以！支持触屏操作（左摇杆移动、右侧滑动转视角、按钮挖放方块）。建议横屏；大屏电脑体验仍然最佳。

## 10. 给老师的课堂建议

- **导览课（15–20 分钟）**：老师建房间投屏领队，学生加入跟随，沿 ①→⑥ 顺序走一遍，每到一个信息点让一位学生朗读讲解；
- **讨论题**：为什么入口不设门？圆形庭院解决了什么问题？绿色屋顶有哪些作用？矩形与圆形相交表达了什么？
- **建造任务**：分组在园区空地上为 Campus 设计「三期扩建」，要求延续放射状布局并至少使用一种绿色建筑元素，最后互相参观点评；
- **跨学科**：几何（圆、环、放射对称）、能源与环境（绿色建筑）、本地企业与工业园区认知（Batu Kawan）。
